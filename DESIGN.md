# 拼音轮盘游戏 - 项目设计文档

## 项目概述

一个互动式的拼音学习轮盘游戏，用户通过控制两个玻璃球在轮盘上滚动来选择声母和韵母。

## 视觉设计

### 轮盘结构
- **外圈（韵母环）**：显示所有韵母选项
- **内圈（声母环）**：显示所有声母选项
- 双圈同心圆设计，外圈较大，内圈较小

### 玻璃球指示器
两个玻璃球作为指针，分别在对应的环上滚动：

| 属性 | 韵母球 | 声母球 |
|------|--------|--------|
| 颜色 | 金黄色（#FFD700 系列渐变） | 银白色（#FFFFFF 系列渐变） |
| 位置 | 外圈韵母环上 | 内圈声母环上 |
| 大小 | 适应外圈格子 | 适应内圈格子 |

### 玻璃球样式
```css
/* 金色玻璃球（韵母） */
background: radial-gradient(circle at 30% 30%, #FFEC8B, #FFD700 40%, #B8860B 70%, #8B6914);
box-shadow: inset -5px -5px 15px rgba(0,0,0,0.5),
            inset 5px 5px 15px rgba(255,255,255,0.5),
            0 0 25px rgba(255, 215, 0, 0.6);

/* 银色玻璃球（声母） */
background: radial-gradient(circle at 30% 30%, #FFFFFF, #E8E8E8 40%, #A0A0A0 70%, #606060);
box-shadow: inset -4px -4px 12px rgba(0,0,0,0.5),
            inset 4px 4px 12px rgba(255,255,255,0.6),
            0 0 20px rgba(255, 255, 255, 0.5);
```

## 交互设计

### 触发方式
- 按空格键开始轮盘旋转
- 同时控制两个球旋转

### 物理滚动效果（核心需求）

1. **启动阶段**
   - 球从静止状态开始逐渐加速
   - 模拟真实物理的惯性效果

2. **滚动阶段**
   - 球在格子上持续滚动
   - 速度保持相对稳定
   - 视觉上应该在扇区之间平滑过渡

3. **减速阶段**
   - 逐渐减速（velocity *= 0.992）
   - 模拟摩擦力效果
   - 减速过程平滑自然

4. **停止位置**
   - **重要**：球必须停在扇区**内部**（30%-70% 位置）
   - 不能停在扇区正中间
   - 不能停在扇区边界上
   - 避免停在视觉上不美观的正中间位置

### 边界震动效果
当球接近扇区边界时（距离边界 < 10%）：
- 产生轻微的震动效果
- 模拟"越过栅栏"的感觉
- 震动幅度约 4px

## 技术实现

### 物理状态管理
```javascript
let ballPhysics = {
    yunmu: { angle: 0, velocity: 0, spinning: false },
    shengmu: { angle: 0, velocity: 0, spinning: false }
};
```

### 动画循环
- 使用 `requestAnimationFrame` 实现流畅动画
- 每帧更新球的位置
- 实时同步球与轮盘的旋转角度

### 停止位置算法
```javascript
// 计算当前扇区
const sectorStart = Math.floor(ballPhysics.yunmu.angle / degPerSector) * degPerSector;
// 停在扇区 30%-70% 位置，避免正中或边界
ballPhysics.yunmu.angle = sectorStart + degPerSector * (0.3 + Math.random() * 0.4);
```
